Methods of binary files

For bynary files (types file, file of T) the following methods are defined:

 
function FilePos: int64;
        Returns the current position of the file pointer in a binary file
function FileSize: int64;
        Returns the number of elements in a binary file
procedure Seek(n: int64);
        Sets the current position of the file pointer in binary file to element number n
procedure Truncate;
        Truncates the binary file, discarding all elements from the position of the file pointer

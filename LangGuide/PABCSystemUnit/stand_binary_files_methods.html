Methods of binary files

For binary files (types file, file of T) the following methods are defined:

 
function FilePos: int64;
        Returns the current position of the file pointer in the binary file
function FileSize: int64;
        Returns the number of elements in the binary file
procedure Seek(n: int64);
        Sets the current position of the file pointer in the binary file to the element of number n
procedure Truncate;
        Truncates the binary file, discarding all elements from the position of the file pointer

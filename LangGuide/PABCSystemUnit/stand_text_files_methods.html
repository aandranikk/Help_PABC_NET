Methods for text files

For text files (type Text) the following methods defined:

 
procedure Writeln(a,b,...);
        Writes values a, b, ... to the file and makes a transition to the next line
function ReadInteger: integer;
        Returns an integer value entered from the text file
function ReadReal: real;
        Returns a value of type real entered from the text file
function ReadChar: char;
        Returns a char value entered from the text file
function ReadString: string;
        Returns a string value entered from the text file without going to the next line
function ReadBoolean: boolean;
        Returns a boolean value entered from the text file
function ReadlnInteger: integer;
        Returns an integer value entered from the text file and moves to the next line
function ReadlnReal: real;
        Returns a value of type real, entered from the text file, and moves to the next line
function ReadlnChar: char;
        Returns a char value entered from the text file, and moves to the next line
function ReadlnString: string;
        Returns a string value, entered from the text file, and moves to the next line
function ReadlnBoolean: boolean;
        Returns a boolean value, entered from the text file, and moves to the next line
function Eoln: boolean;
        Returns True if the end of the line was reached, and false otherwise.
function SeekEof: boolean;
        Skips whitespace, then returns True, if the end of the file is reached
function SeekEoln: boolean;
        Skips whitespace, then returns True if the end of the line in the file is reached
procedure Flush;
        Writes the contents of the file buffer to disk
function ReadToEnd: string;
        Returns the contents of a file from the current position to the end as a string

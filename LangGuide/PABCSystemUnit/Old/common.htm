Common standard procedures and functions

Name and parameters Procedure or function Types of parameters Type of returned value Description
Read(a,b,...) procedure a,b,c - variables of primitive type or type string   inputs values from the keyboard into variables a, b ...
Write(a,b,...) procedure a,b,c - expressions of primitive type, type string or pointers   writes the values a, b ... in the output window
Readln(a,b,...) procedure a,b,c - variables of primitive type or type string   inputs values from the keyboard into variables a, b ..., then ignores all remaining characters in the current entry line. If the procedure parameters are not specified, provides a pause in program execution until you press Enter
Writeln(a,b,...) procedure a,b,c - expressions of primitive type, type string or pointers   writes the values a, b ... in the output window and skips to the next line. If the parameters of the procedure are not specified, it only skips to the next line.
Abs(x) function x - integer, real, complex matches the parameter type returns an absolute value (module) x
Sqr(x) function x - integer, real, complex matches the parameter type returns a square x
Sqrt(x) function x - real, complex matches the parameter type returns the square root of x
Sin(x) function x - real, complex matches the parameter type returns the sine x
Cos(x) function x - real, complex matches the parameter type returns the cosine x
Ln(x) function x - real, complex matches the parameter type returns the natural logarithm of x
Exp(x) function x - real, complex matches the parameter type returns Euler's number e raised to the power of x (e=2.718281...)
Arctan(x) function x - real, complex matches the parameter type returns the arctangent of x
Power(x,y) function x, y - real real returns x raised to the power y
Round(x) function x - real integer returns the result of rounding x to the nearest integer
Trunc(x) function x - real integer returns the integer part of õ
Int(x) function x - real real returns the integer part of õ
Frac(x) function x - real real returns the fractional part of õ
Ord(x) function x - ordinal type integer returns the number of value of ordinal type
Chr(x) function x - integer char returns the character with code õ
Odd(x) function x - integer boolean returns True, if õ is odd, and False otherwise
Inc(x) procedure x - ordinal type   increases x by 1
Dec(x) procedure x - ordinal type   decreases x by 1
Inc(x,n) procedure x - ordinal type,
n - integer		   increases x by n
Dec(x,n) procedure x - ordinal type,
n - integer		   decreases x by n
Pred(x) function x - ordinal type matches the parameter type returns previous value of ordinal type
Succ(x) function x - ordinal type matches the parameter type returns the next value of ordinal type
Random(x) function x - integer integer returns a random integer in the range from 0 to x-1
Random function   real returns a random integer in the range [0..1)
Include(s,x) procedure s - set, x - element of type compatible with the type of base set   includes the x element in the set s
Exclude(s,x) procedure s - set, x - element of type compatible with the type of base set   excludes the x element from the set s

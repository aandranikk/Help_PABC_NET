Common methods for file types

For all standart file types (Text, file, file of T) the following methods are defined:

 
procedure Write(a,b,...);
        Writes values (a, b, ...) to a file
function Name: string;
        Returns the file name
function FullName: string;
        Returns the full file name
function Eof: boolean;
        Returns True, if the end of the file is reached, and False otherwise
procedure Close;
        Closes the file
procedure Erase;
        Deletes the file
procedure Rename(newname: string);
        Renames the file to "newname"

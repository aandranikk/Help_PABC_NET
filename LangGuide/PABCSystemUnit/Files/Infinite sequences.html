Infinite sequences

function Cycle<T>(Self: sequence of T): sequence of T; extensionmethod;
        Repeats the sequence infinitely
function Step(Self: integer): sequence of integer; extensionmethod;
        Returns an infinite sequence of integers that starts with the current value and has a common difference (step) of successive elements equal to 1
function Step(Self: integer; step: integer): sequence of integer; extensionmethod;
        Returns an infinite sequence of integers that starts with the current value and has a common difference (step) of successive elements equal to step
function Step(Self: real; step: real): sequence of real; extensionmethod;
        Returns an infinite sequence of real numbers that starts with the current value and has a common difference (step) of successive elements equal to step

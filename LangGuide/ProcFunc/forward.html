Forward declaration

In some cases it is necessary to call a subroutine that is defined later in the code. For instance, such a need occurs with indirect (mutual) recursion (subroutine A calls subroutine B that in turn calls subroutine A). In this case forward declaration is used. It consists of a subroutine's header and forward keyword. For example:

procedure B(i: integer); forward;
procedure A(i: integer);
begin
  ...
  B(i-1);
end;
procedure B(i: integer);
begin
  ...
  A(i div 2);
end;

It is forbidden to use forward declaration for a function that is already defined.

forward keyword cannot be used with methods. There is no need for that because you can use methods that are defined later in the body of a class.

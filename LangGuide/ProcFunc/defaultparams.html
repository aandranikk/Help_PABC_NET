Default parameters

It is possible to use default parameters in the header of a subroutine. In order to do that the default value of a parameter must be provided as follows:
<parameter identifier>: <parameter type> := <default value>.
If default arguments are omitted while invoking a function then their previously specified default values are used. Note that default parameters must always be passed by value, and they must be defined at the end of a parameter list.

For example:

procedure PrintTwo(a,b: integer; delim: char := ' ');
begin
  write(a,delim,b);
end;

...
PrintTwo(3,5);
PrintTwo(4,2,';');

All available options of subroutine invocation are used to perform overload resolution.

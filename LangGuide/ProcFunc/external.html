Invocation of subroutines implemented in unmanaged DLLs

To call a subroutine implemented in an unmanaged DLL (contains native/machine code instead of .NET-code) the following code is used:

subroutine's header external 'DLL's name' name 'function's name inside the DLL'; 

For example:

function MessageBox(h: integer; m,c: string; t: integer): integer;
  external 'User32.dll' name 'MessageBox';
...
MessageBox(0,'Hello!','Message',0);
